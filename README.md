# 🚀 SkillCraft Technology – Data Analytics Internship

Welcome to my Data Analytics Internship repository at SkillCraft Technology. This repository showcases the projects and tasks completed during the internship, focusing on Excel, Python, Data Cleaning, Exploratory Data Analysis (EDA), and Business Intelligence using Power BI.

## 📌 Internship Overview

The SkillCraft Technology Data Analytics Internship is designed to provide hands-on experience in data analysis, data preprocessing, visualization, dashboard development, and business insights generation through real-world datasets.
